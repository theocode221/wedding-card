Place one JPEG per segment here. Filename = segment id + extension, e.g.:

  anep.jpg   (or .jpeg / .JPG — any of these is fine)
  din.jpg
  hafizi.jpg
  nabil.jpg

These match the segment `id` values in src/data/blessings.ts (case of extension can differ).
Photos appear in the square frame on the result card after a spin (not on the wheel).
